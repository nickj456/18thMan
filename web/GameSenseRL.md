# Game Sense Coaching — Rugby League (18th Man AI Reference)

## Overview

Game Sense coaching is an athlete-centred approach that develops decision-making, awareness, and tactical understanding through game-like situations.

Instead of drills first → game later, you flip it:

**Game → Question → Reflect → Adjust → Replay**

This method aligns with the constraints-led approach in skill acquisition and is the foundation for all AI session suggestions on this platform.

---

## Core Principles

### 1. Learn Through the Game
- Players learn best in realistic scenarios
- Training mimics match conditions
- Decisions matter more than perfect technique

### 2. Guided Discovery (Not Telling)
- Coach asks questions instead of giving answers
- Players solve problems themselves

Examples:
- "What did you see there?"
- "Why did that overlap happen?"
- "What was the defender doing?"

### 3. Decision Making Over Repetition
Focus on:
- When to pass
- When to run
- Where to position
- Not just how to execute skills

### 4. Constraints-Based Coaching
Shape behaviour by changing the game, not stopping it.

Examples of constraints:
- Reduced space
- Limited passes
- Scoring conditions
- Player overloads

### 5. High Engagement
- Minimal standing around
- Constant involvement
- Learning through action

## Session Structure (60–90 minutes)

All AI-generated sessions follow this structure. Timing scales within the 60–90 min window.

| Section | Duration | Purpose |
|---|---|---|
| Warm-Up (game-based) | 10–15 min | Movement, touches, low-intensity game |
| Modified Game 1 (constraint) | 15–20 min | Introduce focus area via constraint |
| Question & Reflect | 2–3 min | Guided questioning — keep it short |
| Modified Game 2 (progression) | 15–20 min | Same game with added challenge |
| Game Sense Competition | 15–20 min | Full game-like competition with scoring condition tied to focus area |
| Review | 3–5 min | Key learning points, take-aways |

## Focus Area Rotation Pool

The AI rotates through these 15 focus areas, one per session, in order of least recently used. Each area is used once before repeating.

### Attack
1. Attacking Width
2. Short Side Attack
3. Dummy Half Play
4. Draw & Pass
5. Support Play & Offloads
6. Set Plays (kick-off, scrums, restarts)

### Defence

11. Completing Sets
12. Kick Chase & Pressure

### Skills in Context
14. Catching & Passing (under pressure)
15. Ball Security (contact & offload decision)

## Practical Examples

### 🟢 Attacking Width

**Game Setup:** 6v5 attacking overload, full width

**Constraint:** Try only counts if scored in wide channel

**Coaching Focus:** Draw and pass, timing, depth

**Questions:**
- "What created space?"
- "When should you pass?"

---

### 🔴 Line Speed (Defence)

**Game Setup:** 6v6 in 30m grid

**Constraint:** Defence scores a point for dominant tackle or forcing error

**Coaching Focus:** First 3 steps, line integrity, communication

**Questions:**
- "How quickly did we get out of the line?"
- "Who was setting the line speed?"

---

### 🔵 Ruck Speed & Awareness

**Game Setup:** 5v5 with quick play-the-ball

**Constraint:** Dummy half must pass within 2 seconds

**Coaching Focus:** Marker recognition, exploiting A/B defenders, decision making

**Questions:**
- "What options did the dummy half have?"
- "Where was the space?"

## Coaching Behaviours

### What You SHOULD Do
- Ask questions
- Let players fail
- Keep games flowing
- Observe patterns

### What You SHOULD AVOID
- Over-coaching
- Stopping play too often
- Giving all the answers
- Isolated drills with no context

---

## Questioning Framework

**Before:** "What are we trying to achieve?"

**During:** "What are you seeing?"

**After:** "What would you change next time?"

---

## When NOT to Use Game Sense

Use direct coaching when:
- Teaching brand new technique
- Safety is involved (e.g. tackling basics)
- Clear correction is needed

Best approach: blend both styles.

## AI Session Generation Instructions

When generating a session suggestion, the AI must:

1. **Check rotation history** — identify which focus areas the group has covered recently and select the least recently used area from the pool above.
2. **Structure the session** to the 60–90 min template above (6 sections).
3. **Design each game with a specific constraint** tied to the focus area.
4. **Include 2–3 guided questions** the coach can ask at the reflect stage.
5. **Scale timing** proportionally if the group's typical session is shorter or longer.
6. **Never repeat a focus area** until all 15 have been used at least once.
7. **Keep coaching language athlete-centred** — frame suggestions as questions and challenges, not instructions.
